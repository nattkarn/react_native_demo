# React Native Demo Expo
## Description
Example
- Splash Screen
- Menu Home Page
- Get Data from Api
- Login with Jwt Api
- Show data from Json api
- QR Code Scaner
- Open Google Map from lat long

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# android
$ npm run android
```
